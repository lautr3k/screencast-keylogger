# screencast-keylogger
Simple screencast keylogger

## Requirements
- Python 2.7
- pyHook (https://sourceforge.net/projects/pyhook/)
- GTK+3 and WebKit+3 (https://wiki.gnome.org/action/show/Projects/PyGObject)

## OS support
For now only for Windows (tested on W10)
